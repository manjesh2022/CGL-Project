# CGL-Project
Computer Graphics Laboratory Project on "ATOM SIMULATION" using OpenGL in C

# To execute follow
1.install g++ compiler & glut library<br/>
2.use g++ atom.c -lGL -lGLU -lglut commond to compile<br/>
3.use ./a.out to see the output
